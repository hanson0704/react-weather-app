# 🌤️ React Weather App

A responsive weather application built using React that fetches real-time weather data for any city using the OpenWeather API.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Real-time temperature display (°C)
* 🌤️ Weather condition with dynamic icons
* 💧 Humidity, wind speed, and pressure details
* 🌅 Sunrise & 🌇 Sunset time calculation
* ⚡ Fast API-based data fetching using Axios
* 🎨 Clean and responsive UI using styled-components

---

## 🛠️ Tech Stack

* **Frontend:** React (Functional Components + Hooks)
* **Styling:** Styled-components
* **API:** OpenWeather API
* **HTTP Client:** Axios
* **Deployment:** GitHub Pages

---

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/hanson0704/react-weather-app.git
```

2. Navigate to the project:

```
cd react-weather-app
```

3. Install dependencies:

```
npm install
```

4. Run the app:

```
npm start
```

---

## 🌐 API Used

* OpenWeather API
* Fetches real-time weather data using city input

---

## 🚀 Deployment

To deploy on GitHub Pages:

```
npm run deploy
```
